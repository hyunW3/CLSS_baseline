# CLSS (Contiunal Learning Semantic Segmentation) baselines
- This is CLSS baseline implementation
- Currently, it includes MiB, PLOP, DKD.

## Environment
- CUDA 11.1
- python 3.8
- The shell code for setting environment is in `scripts/env_create.sh`
